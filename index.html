01 — Project

Portfolio
Optimization

Objective

Construct a high-performance portfolio of 5 stocks (NVDA, MSFT, LLY, JNJ, GD) via mean-variance optimization, targeting the Maximum Sharpe Ratio (MSR) portfolio on the Efficient Frontier — maximizing risk-adjusted return.

The Approach

Formulated a non-linear optimization problem to identify the Tangency Portfolio, where each additional unit of risk yields the highest marginal return. Benchmarked against a naive equal-weight baseline and the Minimum Variance portfolio to evaluate the risk-return tradeoff.

02 — Project

Maritime Freight
Rate Forecasting

Objective

To engineer a high-precision predictive model for shipping rates, enabling data-driven chartering decisions for our client, Eastport Maritime.

The Approach

Developed multiple iterations to improve model prediction, moving from a univariate ARIMA baseline to a multivariate ARIMAX model that incorporates external variables. Conducted extensive stress testing by comparing performance when using "Actual" vs. "Predicted" exogenous variables.

Key Results
  • 6.46% Test MAPE Attained exceptional predictive precision on out-of-sample data, outperforming the 10%–20% industry standard for volatile maritime sectors.
  • 75% Directional Accuracy Established a significant strategic edge by correctly anticipating market changes (Benchmark: 60%–80%).
03 — Project

Neural Network
Verification

Objective

To mathematically verify the robustness of a leaf-vitality classifier, ensuring its reliability against imperceptible environmental changes.

The Approach

Translated a deep learning model into a Mixed-Integer Linear Programming framework to find absolute worst-case vulnerabilities. Tested boundary of ±0.01 brightness change in each pixel to identify if imperceptible pixel changes could flip predictions from "Diseased" to "Healthy".

Key Results
  • Failed Certification Proved the model was not certifiably robust, highlighting critical risks for real-world deployment.
  • Potential Improvements Adversarial training by including perturbed images in the training set to strengthen model resilience.
